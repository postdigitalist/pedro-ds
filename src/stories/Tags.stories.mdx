# ¿Para qué sirven los tags?

Los tags pueden servir para:

- Filtrar resultados de una búsqueda
- Mostrar el status de un elemento
- Destacar a un elemento por su pertenencia a cierta categoría.

# Cómo se construye un tag

Creamos clases para 5 colores de tag:

- Azul / `.tagBlue`
- Gris / `.tagGray`
- Naranja / `.tagOrange`
- Verde / `.tagGreen`
- Rojo / `.tagRed`

Recomendamos usar los tags verdes y rojos de forma particularmente cuidadosa.

# Prácticas sugeridas para el uso de íconos

Podes incluír íconos dentro de los tags. Recomendamos:

- Que su tamaño sea entre `w-2` y `w-4`
- Que tenga `mr-1` de margen
- Que no se trate de un ícono muy complejo y que aluda a la acción que el usuario puede llevar a cabo con respecto al tag, que no sea meramente ilustrativo
- Si el usuario no puede interactuar con el tag, no debería incluír un ícono

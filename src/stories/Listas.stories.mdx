# Todo lo que necesitas saber sobre listas

Este es uno de los componentes más sencillos de nuestro sistema.

Tiene dos variaciones:

- Lista numerada
- Lista sin numerar

## Nuestras listas sin numerar
Nuestras listas sin numeración se ven así:

- Durazno
- Manzana
- Banana
- Tomate 
- Lechuga

Nuestras listas numeradas se ven así:

1. Garbanzos
2. Anchoas
3. Lentejas
4. Acelga

Para crear una lista de acuerdo a los estándares de Pedro, no necesitas añadir ninguna clase. Nuestras listas son lindas por defecto.
